# 💼 AI-Based Resume Analyzer & Shortlisting System

## 📌 Overview

This project is a simple Python-based system that analyzes a candidate’s profile and checks their suitability for a selected job role. It works like a basic hiring system where different factors such as skills, CGPA, projects, and internship experience are considered.

The system also provides feedback and suggestions to help the user improve their profile.

---

## 🎯 Problem Statement

Many students are unsure whether their current skills and profile are enough for a particular job role. They also don’t know what exactly they need to improve.

This project aims to solve that problem by analyzing a candidate’s profile and giving clear feedback.

---

## 🚀 Features

* Role-based analysis
* Skill matching system
* Score calculation (out of 100)
* Candidate classification (Strong / Average / Needs Improvement)
* Missing skills detection
* Improvement suggestions
* Alternative role suggestion based on skills

---

## ⚙️ How It Works

1. User enters:

   * Skills
   * CGPA
   * Number of projects
   * Internship experience
   * Desired job role

2. The system:

   * Matches skills with required job skills
   * Calculates a score based on different factors
   * Decides whether the candidate is selected or not

3. Output includes:

   * Score
   * Skill match percentage
   * Missing skills
   * Suggestions for improvement
